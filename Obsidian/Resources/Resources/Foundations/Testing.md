  - TDD: Test-Driven Development
    - Các step:
      1.  Viết test trước: -> test sẽ fail vì chưa có code.
      2.  Viết code để cho các test pass:
      3.  Refactor: Cải tiến code, đảm bảo vẫn pass.
    - Thường sẽ áp dụng Unit test.
  - TLD: Test-Last Development
